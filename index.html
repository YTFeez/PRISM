KASHOW est notre robot pour le concours Paul Cornu 2026 — navigation autonome, obstacles et précision de trajectoire.

Passage d’une base mBot à un châssis plus personnel type voiture de course, identité forte et mécanique pensée pour la piste.

Ultrasons, IR sol, luminosité ; code modulaire pour s’adapter aux consignes. Moteurs avec encodeurs, LiPo, firmware orienté réactivité.
